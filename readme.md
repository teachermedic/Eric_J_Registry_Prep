🩺 Eric J's Field Notes

The High-Velocity AEMT & Paramedic Clinical Engine

Eric J's Field Notes is a mobile-first, high-performance study platform designed for Advanced EMT and Paramedic students. It bridges the gap between basic registry prep and elite-level clinical pathophysiology.

🚀 View the Live App

(https://teachermedic.github.io/Eric_J_Registry_Prep/)

⚡ Key Features

Study vs. Exam Modes: Toggle between a low-pressure "Review" environment and a high-stakes, timed "Exam" simulation.

"Make Me a Physician" (Elite Study): A specialized module containing deep-dive pathophysiology and USMLE-style high-yield clinical pearls.

Active Recall Logic: Features a custom "Open-Review" engine that forces students to mentally formulate clinical findings before revealing the "Clinical Truth."

MasterData Dashboard: Fully integrated with Google Sheets and Apps Script to track student performance, category mastery, and study trends in real-time.

Progressive Web App (PWA): Install the engine directly to your mobile device for a native app experience on the truck or in the classroom.

Dark Mode Enabled: Optimized for 24-hour shifts with a specialized low-light theme.

📊 The "MasterData" Integration

This project doesn't just quiz students; it tracks them. Using a custom Google Apps Script bridge, every session is logged into a MasterData Spreadsheet, allowing instructors to:

Monitor student engagement (Review vs. Exam volume).

Identify specific "Knowledge Gaps" in categories like Cardiology or Pathophysiology.

Track mastery percentages over time.

🛠️ Built With

Frontend: Vanilla JavaScript, HTML5, CSS3 (Custom Variables).

Backend: Google Apps Script (Web App API).

Data: Google Sheets (MasterData v2).

UI/UX: Material Icons, Google Fonts, LaTeX (via MathJax).
